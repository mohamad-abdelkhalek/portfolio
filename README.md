# My Simple Portfolio

Welcome to my portfolio project! This is a simple personal website showcasing my education, activities, and contact information.


---

## 📄 Features
- **Responsive Design**: Adapts to different screen sizes for better accessibility.
- **Navigation Bar**: Quick access to different sections of the portfolio.
- **Content Sections**:
  - **Education**: Displays academic background and timeline.
  - **Activities**: Lists hobbies and professional engagements.
  - **Contact**: Links to GitHub and LinkedIn profiles.
